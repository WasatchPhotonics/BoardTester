Instructions
Go to http://waspho.com Click the Get Started button to view the list of current checkouts.
Click the users link on the bottom navigation bar to begin the login process. Only users with Wasatch Photonics domains are authorized to add and modify checkouts.
Review your current login status, then click the link to login.
The CookBook uses Google's oauth protocol to authenticate users. Click Allow to permit the CookBook to see your email address. This is only necessary for Wasatch Photonics users looking to add or modify checkouts.
Verify your current login status. Click the Checkouts link to view the list of current checkouts and add a new item.
The Checkouts list now includes an 'Add Checkout' button, available only to users at Wasatch Photonics domains. Click the Add Checkout button.
Fill out the new checkout form with the unique serial number. Enter the calibration coefficients determined during the manufacturing procedure. Any non-blank value is acceptable for the calibration coefficients, and will be included as is in the calibration report. Click the submit button.
The image here is an example calibration report generated. Note how the calibration report is populated with the submitted values. Click the QR Label image, and print it on a QL-700 label printer for best results. The checkout has now completed it's bare minimum processes. Note how the product imagery is blank, and there is no software to run the device associated with this checkout. The following steps show to add these files to a checkout.
Scroll down on the checkout page, and click Add File.
On the file add page, choose the appropriate file descriptor before uploading a file. In the example to the right, the "High Camera" type is designated. Use the "Default" option for control software, instructions files, and other file types that are not expected to fill a role. Use "High Camera" for a JPG image that is created from a camera mounted high above the device. Use "Low Camera" for a JPG image that is created from a camera mounted at device level. Use "Work Ticket" for scanned PDF's.
Shown to the right is an example of both high and low camera imagery populated in an example report.
Click "Add File" again, upload the work ticket.
Click "Add File" again, upload the Dash-setup.exe file, if relevant for this device. Repeat this process with instructions and any other files the customer needs access to.
The image to the right shows the results of the work ticket and Dash executable file uploading.
